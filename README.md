# Приложение для составления расписания задач на неделю

#### Для построения приложения выполните следующие действия:

~~~~
1. Установка зависимостей проекта: 
    yarn install
~~~~

~~~~
2. Для запуска приложения в режиме отдалки: 
    2.1. Платформа Android: "expo start --android"
    2.2. Отладка в браузере Android: "expo start --web"
~~~~

~~~~
3. Для компиляции приложения Android: 
    3.1. Необходим аккаунт на https://expo.dev/ (доступы к нему будут запрошены на этапе сборки)
    3.2. В аккаунте нужно настроить проект имя которого соответствует значению "slug" из файла app.json
    3.3. Запустить команду expo build:android
~~~~
